# 🎲 TÀI XỈU - Telegram Mini App

Mini App chơi Tài Xỉu (Sic Bo) trực tiếp trên Telegram với đồ họa 3D đẹp mắt.

## ✨ Tính năng

- 🎲 Xúc xắc 3D chân thực
- 💰 Hệ thống số dư ảo
- 📊 Lịch sử chi tiết
- 🏆 Bảng xếp hạng
- 🎨 Giao diện Glassmorphism
- ⚡ Animation mượt mà

## 🚀 Cài đặt Bot Telegram

1. Tạo bot qua [@BotFather](https://t.me/botfather)
2. Gửi lệnh `/setmenubutton`
3. Chọn bot của bạn
4. Nhập URL: `https://TEN_GITHUB.github.io/taixiu-telegram-miniapp/`
